# 豬肉泡菜

## 食材
* 豬五花肉
* 蒜頭
* 韓式泡菜
* 韓式辣醬
* 洋蔥
* 青菜

### 調味料
* 香油
* 米酒

## 步驟
1. 洋蔥順紋切絲, 切蒜末
2. 洋蔥, 蒜末, 豬肉, 韓式泡菜, 韓式辣醬放入碗中. 淋上香油, 醬油. 攪拌
3. 熱鍋, 放點香油, 放入肉
4. 煮到差不多時,加點米酒, 跟你想吃的青菜
5. 最後補點水, 讓泡菜跟著肉的味道出來
