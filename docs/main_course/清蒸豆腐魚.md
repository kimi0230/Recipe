# 清蒸豆腐魚

## 食材
* 魚
* 豆腐
* 薑
* 蔥

### 調味料
* 鹽
* 米酒
* 醬油

## 步驟
1. 魚片沖水洗淨後用廚房紙巾擦乾抹上鹽
2. 魚片上放上薑片、蔥段
3. 淋上醬油、少許米酒後放入電鍋(外鍋+1杯水)
4 .跳起後再放入蔥絲

## 參考
鱸魚等其他魚也可以
豆腐放於魚下面
魚片可切小塊
醬油可用鰹魚醬油
醬油、香油可起鍋後淋
