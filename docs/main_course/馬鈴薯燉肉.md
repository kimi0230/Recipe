# 馬鈴薯燉肉

## 食材
* 豬肉
* 馬鈴薯
* 紅蘿蔔
* 洋蔥

### 調味料
* 醬油 2 匙
* 米酒 2 小匙
* 味淋 1 小匙

## 步驟
1. 洋蔥切絲, 馬鈴薯, 紅蘿蔔切塊
2. 熱鍋後放入肉片跟洋蔥, 炒到豬肉跟洋蔥微焦, 再放入馬鈴薯, 紅蘿蔔.
3. 翻炒後, 倒入電鍋, 加入調味料
4. 加水至稍微淹到食材
5. 電鍋外鍋放一杯水
