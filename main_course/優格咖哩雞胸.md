# 優格咖哩雞胸

## 食材
* 雞胸肉
* 咖哩塊或咖哩粉
* 黑胡椒
* 優格


## 步驟
1. 冷開水+鹽+優格 將雞胸肉放入. 泡一晚, 最少要一小時
2. 將雞胸肉擦乾, 兩面都塗抹上黑胡椒, 咖哩粉或塊(想辦法磨成碎碎的)
3. 放入鍋中用大火煎至表面上色
4. 小火煎並蓋上蓋子, 煎約5分鐘.
5. 熄火讓雞胸悶5分鐘.
