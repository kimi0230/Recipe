# 漢堡肉

## 食材
* 豬絞肉(細) 250g
* 雞蛋一顆
* 牛奶20cc
* 麵包粉10g
* 海鹽、黑胡椒適量


## 步驟
1. 將牛奶與麵包粉混合，浸泡三分鐘
2. 將雞蛋、浸泡麵包粉的牛奶、海鹽、黑胡椒一起放入缸盆中。
3. 先將絞肉用打水的方式，把液體都吸收到絞肉中。
4. 另外將絞肉摔打數次至有黏性。
5. 放入冰箱冷藏一小時
6. 加入炒過的洋蔥末，即可放入鍋中煎熟。

## 祕訣
1. 將牛奶與麵包粉浸泡至濕潤再放入絞肉中，可以讓漢堡排吃起來更鬆軟。
2. 將洋蔥丁用奶油炒至焦化，需要10分鐘以上，再將炒過的洋蔥放入絞肉中，可以增加漢堡排的香氣，
3. 放入平底鍋中油煎的時候，記得要加鍋蓋，漢堡肉有一定的厚度，加上蓋子可以讓漢堡肉有肉汁。
