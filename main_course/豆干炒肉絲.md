# 豆干炒肉絲

## 食材
* 肉絲
* 豆乾
* 蔥
* 蒜

## 醃肉
* 肉絲用醬油、米酒、香油依序抓勻醃15分後用太白粉抓勻

### 調味料
* 醬油膏
* 糖


## 步驟
1. 豆干切絲or片
2. 先炒豆干絲煎上色後取出
3. 再將肉絲炒至6分熟後加入蒜末、辣椒絲、蔥白段爆香
4. 放入豆干絲後加入醬油膏、水、糖，炒至醬汁收汁
5. 起鍋前放入蔥綠段即可
