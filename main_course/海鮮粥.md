# 海鮮粥

## 食材
* 薑
* 米
* 海鮮(魚蝦)

## 步驟
1. 先講米洗好放入冷凍
2. 薑切成絲
3. 米跟水 1:15
4. 中火, 米 水 薑絲放入煮, 煮到你想要的程度
5. 放入青菜 海鮮
6. 加鹽
7. 加蛋
