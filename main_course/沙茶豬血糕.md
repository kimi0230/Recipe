# 沙茶豬血糕

## 食材
* 豬血糕
* 杏鮑菇
* 九層塔
* 蔥
* 蒜
* 辣椒

### 調味料
* 沙茶醬
* 米酒
* 醬油

## 步驟
1. 豬血糕切塊、表面煎香後盛起
2. 杏鮑菇滾刀切、煎香後盛起
3. 煸薑片、香氣出來後放入蒜、辣椒炒香
4. 放豬血糕、加入沙茶、米酒、醬油、水，滷至醬汁微乾
5. 再放入杏飽菇炒1分、起鍋前放入蔥段、九層塔
