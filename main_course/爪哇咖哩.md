# 爪哇咖哩

## 步驟
1. 拌炒
先用油熱鍋, 再將切成適當大小的肉塊及蔬菜下鍋
2. 加水煮沸
加水煮沸(1300 ml), 並撈起湯上浮沫, 以小火~中火燉煮至材料約 15分鐘
3. 加入咖哩塊
暫停熄火, 撥開咖哩塊放入攪拌至溶化
4. 燉煮
再以小火燉煮至濃稠狀, 即可食用 10分鐘
